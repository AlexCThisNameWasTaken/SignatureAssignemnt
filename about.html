About This Site

The Bird Web is a student made project created to sample and explore web devlopment and display said skills while also showcasing information about birds.

Mission

My mission is to educate and inspire interest in bird species through a clean, beginner friendly website layout. Although I chose an informative website about birds from a list of available options, I additioinally chose a bird informational for my love for biology and sciences.

About the Creator

This site was created by Alexandru Catan, a student and Loveland Highschool learning basics of web development in Fundamentals of Web Development 1. This project serves as a showcase of the skills that I have accumulated throughout my time learning web development.

What's on the site?

  • Home: Introduction to the bird Web
  • Resources: Links, videos, and other bird-related content
  • Contact: A form for feedback or comments
  • The Bird Web: Web page connecting various birds across the bird informational pages
